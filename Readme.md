# TeclasMnuATexclasMnuXml

Este repositorio contiene programas para convertir archivos de configuración de teclas (TECLAS.MNU) de versiones antiguas de Digi al formato TECLAS.MNU.XML.

El formato TECLAS.MNU.XML soluciona los siguientes problemas:

* El formato TECLAS.MNU no había ninguna manera de saber a qué pulsación de tecla se correspondía el identificadore de tecla.
* El formato TECLAS.MNU no permitía añadir más de una orden a una pulsación de tecla, obligando a crear archivos de macroinstrucciones (arrobas) en caso de querer asignar más de una orden a una pulsación de tecla.
* El formato TECLAS.MNU no permitía añadir una descripción que explicase el comportamiento de pulsar la tecla.
* El formato TECLAS.MNU no permitía imprimir la tabla de forma tabular.
