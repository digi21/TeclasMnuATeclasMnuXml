# CreaDiccionarioTeclas

Este programa MFC (en lenguaje C++) replica el algoritmo que utilizaban Digi3D 2000, 2002, 2003, 2005 y 2007 para generar un código de tecla (que se almacenaba en archivos TECLAS.MNU) y genera un archivo con código en C# que se utiliza en el programa TeclasMnuATeclasMnuXml como diccionario.

Es un programa que muestra un cuadro de diálogo e invita al usuario a pulsar combinaciones de teclas. Estas combinaciones se van almacenando en memoria y al pulsar sobre el botón Aceptar se solicita la ruta del archivo a crear y se crea un archivo con el código correspondiente en C#.
